# Examen Final Diseño Web - Eco Pilgua

## 📖 Descripción del Proyecto
Este repositorio contiene el desarrollo de la aplicación web para el emprendimiento **Eco Pilgua**, diseñada como parte del Examen Final de la asignatura de Diseño Web. El proyecto ha sido construido utilizando tecnologías open source, con el objetivo de estructurar, dar estilo e implementar eventos web que respondan a las necesidades del negocio y cumplan con los estándares de diseño responsivo y semántica web.

## 🚀 Despliegue (Deploy)
La plataforma se encuentra publicada y completamente accesible al público a través del siguiente enlace de producción:
* **Sitio en vivo:** [https://endearing-parfait-a79e8b.netlify.app/](https://endearing-parfait-a79e8b.netlify.app/)

## 🛠️ Tecnologías Utilizadas
* **HTML5:** Estructuración semántica del contenido para garantizar accesibilidad.
* **CSS3:** Estilos personalizados, maquetación y diseño fluido adaptable a dispositivos móviles y de escritorio.
* **JavaScript:** Implementación de interactividad, control de flujos de navegación y gestión de eventos dinámicos en el cliente.

## 📁 Estructura del Proyecto
La organización del código fuente sigue las pautas tradicionales de arquitectura web limpia:
* `index.html`: Archivo principal y punto de entrada al sitio web.
* `assets/`: Directorio que almacena recursos estáticos como imágenes, hojas de estilo CSS y scripts de lógica propia.
* `vendors/`: Carpetas destinadas a librerías externas de terceros utilizadas para extender las funcionalidades visuales o estructurales.
